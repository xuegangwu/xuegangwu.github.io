## ⏳ 浏览器缓存

### 如果看不到 favicon，请执行：

**1. 强制刷新**
```
Windows/Linux: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

**2. 清除浏览器缓存**
- Chrome: 设置 → 隐私和安全 → 清除浏览数据
- 选择"缓存的图片和文件"
- 点击"清除数据"

**3. 关闭浏览器重新打开**

**4. 访问页面**
```
https://xuegangwu.github.io/
```

**5. 查看 Tab 页**
- 应该显示 🦞 龙虾 icon
