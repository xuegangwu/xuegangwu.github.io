# 光储充一体化

## 概念

光储充一体化是指将光伏发电、储能系统和充电设施（主要为电动汽车充电桩）有机整合，形成一套完整的"发-储-充"能源体系。

```
光伏发电 → 储能系统 → 充电桩
    ↑           ↓
    └──────调配─────┘
```

## 核心优势

- **自发自用**: 光伏发电优先供充电桩使用，余电储存
- **峰谷套利**: 利用储能系统在谷时储电、峰时放电
- **稳定供电**: 储能平抑光伏波动，提升充电稳定性
- **减排效益**: 最大化使用清洁能源

## 应用场景

1. **工业园区**: 工厂屋顶光伏 + 储能 + 职工充电桩
2. **商业综合体**: 停车场光伏棚 + 储能 + 快充站
3. **社区/小区**: 共享充电桩 + 家用储能
4. **高速服务区**: 大型光储充站
