# 分布式光伏补贴政策

## 核心内容

| 项目 | 标准 |
|------|------|
| 补贴标准 | 0.05 元/千瓦时 |
| 适用范围 | 装机容量 ≤6MW 的分布式项目 |
| 发放方式 | 按实际发电量结算 |
| 有效期 | 2026年1月1日起 |

## 政策背景

为促进分布式光伏健康发展，国家能源局自2026年起实施新的补贴标准。

## 政策演变

- **2024-2025**: 初始补贴标准 0.10 元/千瓦时
- **2026-01-01**: 补贴退坡至 0.05 元/千瓦时
- **2027 (预期)**: 预计补贴完全退出

> [!open]
> - 2026年Q2是否会进一步退坡？待确认
> - 2027年补贴是否完全退出？
