- [ ] LangChain: `pip install langchain langchain-community`
- [ ] API Keys（至少一个）: OpenAI / Anthropic / Groq
- [ ] VS Code 或 Cursor（推荐 cursor）
- [ ] Git + GitHub CLI

### 推荐工具
- [ ] Obsidian（笔记管理）
- [ ] Notion（任务追踪）
- [ ] Raycast（效率工具，macOS）

## Week 1 — LangChain 入门

### 目标
跑通 LangChain 核心概念：Prompt Template → Model → Output Parser → Chain

### 每日任务

#### W1D1（周一）— 概念扫盲
**早上 1h：** 通读 [LangChain Quickstart](https://python.langchain.com/docs/get_started/quickstart)
**早上 1.5h：** 视频（选择其一）：
- [ ] [LangChain 官方 YouTube - Getting Started](https://www.youtube.com/playlist?list=PLqZXAkvF1tPNfGKz1jtiDcPOcDZspVl9_)（推荐）
- [ ] [ChatGPT + LangChain 教程（B 站）](https://www.bilibili.com/)

**晚上 2.5h：**
- [ ] 搭建项目目录 `~/ai-learning/agent-harness/`
- [ ] 创建虚拟环境：`uv venv .venv && source .venv/bin/activate`
- [ ] 安装依赖：`pip install langchain langchain-openai python-dotenv`
- [ ] 创建 `.env` 文件，写入 `OPENAI_API_KEY=sk-xxx`
- [ ] 跑通 Quickstart 前 3 个例子（LLMChain、ChatModel、PromptTemplate）

**交付物：** `w1d1_hello_chain.py`

---

#### W1D2（周二）— Prompt Template + Output Parser
**早上 1h：** 精读 [Prompt Template 文档](https://python.langchain.com/docs/modules/model_io/prompts/)
**早上 1.5h：** 研究 [Output Parser 文档](https://python.langchain.com/docs/modules/model_io/output_parsers/)

**晚上 2.5h：**
- [ ] 实现一个结构化输出 Parser（JSON / Pydantic）
- [ ] 做一个"简历优化"Agent：输入职位描述，输出优化后的简历要点

**交付物：** `w1d2_resume_agent.py`

---

#### W1D3（周三）— Chain（链式调用）
**早上 1h：** 读 [LCEL（LangChain Expression Language）](https://python.langchain.com/docs/expression_language/)
**早上 1.5h：** 画 LCEL 链的流程图（手绘或 Excalidraw）

**晚上 2.5h：**
- [ ] 用 LCEL 重构 W1D2 的简历 Agent
- [ ] 加上 chain.invoke() 和 stream() 两种调用方式
- [ ] 添加简单的 callback（打印 token 消耗）

**交付物：** `w1d3_lcel_chain.py`

---

#### W1D4（周四）— Memory（记忆）
**早上 1h：** 读 [Memory 文档](https://python.langchain.com/docs/modules/memory/)
**重点：** ConversationBufferMemory / ConversationSummaryMemory

**晚上 2.5h：**
- [ ] 给 W1D3 的链加上 ConversationBufferMemory
- [ ] 实现一个可对话的 CLI 机器人，支持多轮上下文

**交付物：** `w1d4_conversation_bot.py`

---

#### W1D5（周五）— Tool（工具调用）
**早上 1h：** 读 [Tools 文档](https://python.langchain.com/docs/modules/tools/)
**重点：** `@tool` decorator、Wikipedia / DuckDuckGo / Calculator

**晚上 2.5h：**
- [ ] 给对话机器人加 2-3 个 Tool（搜索、计算、日期）
- [ ] 实现一个"助理 Agent"：能查天气、算数学、搜 Wikipedia

**交付物：** `w1d5_tool_calling_bot.py`

---

#### W1D6（周六）— 整合 + CLI 项目

**全天 5h：**
- [ ] 把 W1D1~W1D5 的代码整合为一个 CLI 聊天机器人
- [ ] 支持：对话历史、工具调用、结构化输出
- [ ] 用 argparse 或 typer 做 CLI 界面
- [ ] 写 README.md（功能说明 + 使用方法）

**交付物：** `w1_cli_chatbot/` 项目目录，GitHub 私有仓库初始化

---

#### W1D7（周日）— 复盘 + 整理笔记

**全天 5h：**
- [ ] 整理一周学到的东西 → Obsidian 或 Notion
- [ ] 画一张 LangChain 核心概念图（保存到 `docs/` 目录）
- [ ] 列出 3 个不懂的点，标记 W2 重点研究
- [ ] 休息 + 轻松看视频

**本周交付物清单：**
- [ ] `w1d1_hello_chain.py`
- [ ] `w1d2_resume_agent.py`
- [ ] `w1d3_lcel_chain.py`
- [ ] `w1d4_conversation_bot.py`
- [ ] `w1d5_tool_calling_bot.py`
- [ ] `w1_cli_chatbot/`（完整项目）
